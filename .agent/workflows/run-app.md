---
description: How to run the Heart Attack Risk Prediction System
---

1. Open a terminal in the project directory: `C:\Users\OM AHIR\.gemini\antigravity\scratch\heart-attack-prediction`
2. Ensure dependencies are installed:
   ```bash
   pip install -r requirements.txt
   ```
3. Train the model (if not already done):
   ```bash
   python train_model.py
   ```
4. // turbo
   Run the Streamlit web application:
   ```bash
   streamlit run app.py
   ```
